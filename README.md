# NHostAPI(alpha-version 0.0.3)

## Termux / Debian minimal sometimes requires
```bash
git clone https://github.com/Karnikhil90/NHostAPI.git
cd NHostAPI
pip install --break-system-packages --user -r requirements.txt
python3 run.py
```

# Windows Systems
```bash

git clone https://github.com/Karnikhil90/NHostAPI.git
cd NHostAPI
python --version
pip install -r requirements.txt
python run.py
```
