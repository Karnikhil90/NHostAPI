### ü™ü Windows Setup (Easiest Way)
If you are on Windows, just follow these three steps:

1. **Install Python:** Open your terminal (PowerShell) and paste this:
```powershell
   winget install --id=Python.Python.3.13
```

2. **Download & Extract:** Download the zip from the latest release.
3. **Run:** Double-click the `start.bat` file.

### üíª Developer Setup (Windows / Linux)

If you already have **Git** and **Python** installed:

```bash
# 1. Get the code
git clone [https://github.com/Karnikhil90/NHostAPI.git](https://github.com/Karnikhil90/NHostAPI.git)
cd NHostAPI

# 2. Install dependencies
pip install -r requirements.txt

# 3. Launch
python run.py

```



### üì± Termux / Debian Minimal

```bash
git clone [https://github.com/Karnikhil90/NHostAPI.git](https://github.com/Karnikhil90/NHostAPI.git)
cd NHostAPI
pip install --break-system-packages --user -r requirements.txt
python3 run.py
```
